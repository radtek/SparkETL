View Data History

View Data History  is a replay feature for displaying , data records without re-running the job itself. One can see data records of current run as well as for previous run.

To View/Watch Data History:

  • To view Data History the job should be run at least once in debug mode and watch points should be added on links.

  • To get view Data History option right click on link , where watch point was added. Here we have View Data option. We can either view data for current run or for previous run.

    • Current Job Run: Records displayed are for current run.

    • Prior Job Run: A window is displayed having list of prior job runs.
