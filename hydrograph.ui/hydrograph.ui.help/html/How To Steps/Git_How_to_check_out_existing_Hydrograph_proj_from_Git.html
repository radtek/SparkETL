How to check out existing ELT projects from Git Repository

Steps 1-6 will clone the Git Repository on your local machine.

1) Right click in project explorer > Import

3)select Git > Projects from Git >Next >Select Clone URI and click Next

4) Provide details for Source Git Repository and click Next

5) Select appropriate Git branch and click Next

6) Provide Local folder path and click Next

7) Select import existing eclipse Projects and click Next

Selecting this option will only work if you have checked in .project file. Otherwise, select 'Import as general project'

8) It will show the projects that will be imported in Hydrograph. Click Finish
