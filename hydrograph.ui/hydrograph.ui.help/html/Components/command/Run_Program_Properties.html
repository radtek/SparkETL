Run Program Properties

Properties for the Run Program component can be viewed by Double click component on canvas. The properties contain a 'General' tab. Common and mandatory properties are present in the General tab.

General Properties

Details

  • Name - The Name field will specify the name of the component as desired by the user.
  • ID - ID field will specify unique id for every component.
  • Type - Type further categorizes the component to detailed operation that it intends to perform. Here it is of type Output Run Program.

Configuration

  • Executable Command - Executable Command simply accepts the String in text box.
  • Batch - Batch simply accepts a numeric number starting from 0 to 99 and signifies the batch this component will execute in. By default this is 0.

Validations

The Run Program components applies validations to the mandatory fields. Upon placing the Run Program component on job canvas for the first time (from component palette), the component shows up a warning icon as mandatory properties are not provided.

The properties window also displays error icon on mandatory fields if it has an incorrect value. The error icon is displayed on the tab as well, if any of the field within the tab has some error.

If the properties window has some error even after user visit's it once, then the error icon on the Run Program component on the job canvas changes to error icon. This error icon is removed only when all the mandatory fields are supplied with correct values.
