Cumulate

Cumulate -  Cumulate as name suggests generates an increasing summary of results.It performs cumulative aggregations on input groups of data as defined by the user. It contains one input port and one output port and can be easily distinguished in the component palette.

User is provided with options to either create a custom Java class that specifies the cumulate operation or select a pre-defined standard Operation Class.

The Cumulate component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
