Normalize Mapping View

Hydrograph Release Version 1.0

Mapping view for the Normalize component can be viewed by clicking on the Edit button for Normalize property on General tab. Input fields arriving from the input port is visible at the extreme left of the grid.

All the Input Fields  arriving from the input port are visible on the left side of settings.

All the Output Fields  leaving from the output port are visible on the right side of settings. After changes to operation control and map fields changes in output fields can be viewed by clicking on the Pull From Schema Tab  button.

The center of the settings window has space for operation controls. The Plus sign adds an operation window. The X sign brings up window to choose which existing operation to delete.

Picture of an example Operation control. Operation control has option to choose input fields to work on.
