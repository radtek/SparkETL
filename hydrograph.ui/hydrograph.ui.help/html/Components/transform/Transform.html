Transform

Transform component provides transform operations on the input. It is present under Transform category in the component palette. Transformations are performed on groups of data as defined by the user. It contains one input port and one output port.

User is provided with options to create either a Java class using Operation Class or a expression using Expression Editor that specifies the transform.

The Transform component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
