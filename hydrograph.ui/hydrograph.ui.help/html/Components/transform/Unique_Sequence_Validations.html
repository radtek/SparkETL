Unique Sequence Validations

Hydrograph Release Version 1.0

Validations for the Unique Sequence component are in place for each property present on the Properties window.

General Properties:

  • Sequence Field - Sequence Field is a mandatory field and needs have a valid label. Special characters other than "_" are not allowed for the Sequence Field name. If absent, error for empty field appears along with a message on tool tip for user to fill it appropriately.Validations are in place for valid labels.

  • Phase -Phase has a default value of 0. However, blank text box results in error validation and is not allowed.
