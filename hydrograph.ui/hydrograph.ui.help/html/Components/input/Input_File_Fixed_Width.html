Input File Fixed Width

Input File Fixed Width as the name suggests accepts a fixed width file as input source and is present under the Input category in the component palette.

This component is used to read a fixed width text file. It accepts the length for each record field from the user in the component properties and has a single output port. The execution fails if the accepted record field length does not match with the actual length.

The Input File Fixed Width component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
