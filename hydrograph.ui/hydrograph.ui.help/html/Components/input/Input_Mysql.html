Input Mysql

Input Mysql as the name suggests accepts as records from source table and is present under the Input category in the component palette.

This Component unloads records from either source table or an SQL statements that will select data from one or multiple tables for the further processing of particular job.

The Input Mysql component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
