Output Hive Text File

Output Hive Text File as the name suggests creates Hive table text file as output. It is present under the Output category in the component palette.

This component is used in jobs which write Hive table file as output. Component properties accepts hive database name, table name and partition key names on which the table is partitioned. It has a single input port.

The Output Hive Text File component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
