Output Mysql

Output Mysql as the name suggests write records in Mysql table as output. It is present under the Output category in the component palette.

This Component will load data to MySQL database in required destination table. The records can be new inserts, replace or update to the target table..

The Output Mysql component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
