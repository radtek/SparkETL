Output File Parquet

Output File Parquet as the name suggests writes a parquet file as output and is present under the Output category in the component palette.

This component is used to write a parquet file. It has a single input port.

The Output File Parquet component can be easily distinguished in the component palette.

An enlarged version of the same is depicted when dragged on the canvas.
